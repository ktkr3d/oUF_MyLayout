# [oUF_MyLayout](https://github.com/ktkr3d/oUF_MyLayout)

## Introduction

oUF_MyLayout is a unit frame addon.

## Prerequisite Addons

- [Ace3](https://www.curseforge.com/wow/addons/ace3)
- [LibSharedMedia-3.0](https://www.curseforge.com/wow/addons/libsharedmedia-3-0)

## Download

- [oUF_MyLayout](https://github.com/ktkr3d/oUF_MyLayout/archive/refs/heads/main.zip)

## Configuration

- Slash Command

  ```text
  /mylayout config
  ```

- Game Menu

  <kbd>Esc</kbd> > Options > AddOns > oUF_MyLayout
